# Causal Inference in Marketing Spend

## What This Project Does
Measures the TRUE causal impact of marketing campaigns
using Bayesian Structural Time Series counterfactual modeling.

## Live Demo
https://YOUR-LIVE-URL-HERE.streamlit.app

## Tech Stack
Python, Streamlit, Pandas, Scikit-learn, Statsmodels

## Key Findings
- Radio has highest ROI
- Newspaper has zero ROI — budget should be reallocated
- Campaign effect measured week by week

## Project Structure
- notebooks/ — Analysis notebooks
- src/app.py  — Streamlit dashboard
- data/       — Dataset
- reports/    — Saved charts
